PORTFOLIO

"Alexa, Rap for Me"

At HackNYU 2017, my friends and I created a custom Amazon Alexa skill that supports real-time "freestyle rapping" using natural language processing (NLP).

We wrote the skill as an AWS Lambda function in Python. Check out the code in GitHub. For a more detailed report on how it works and how we put it together, check out our write-up in Devpost.

The N-Robot Formation and Path Tracking Procedure

For our signals and systems course, my teammates and I wrote a MATLAB program to simulate various scenarios where "N" randomly located autonomous robots must locate each other and then follow a desired path of waypoints.

In the video demonstration, four robots line up and follow each other along a series of waypoints using the pure pursuit algorithm with an adaptive lookahead distance. For the implementation details, check out the code and technical report in GitHub.
